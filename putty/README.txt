Use following repo to change theme color for Putty: https://github.com/AlexAkulov/putty-color-themes
* Perfered color is "Desert"

Instructions
1) Clone the repo on computer with Putty
2) Double-click on the registery file you want to use, this will prompt windows to add it to the registery
3) Drag and drop in the same folder to the file called _puttycolor.js
